# Jelve Website – Recreated Version

## Overview

This project is a complete recreation of the Jelve website.  
It includes all core pages and functionality from the original design:

- Home page  
- Contact page  
- Login page (Admin Panel)  
- Editable text sections  
- Lightweight data storage for admin access  

**Live URL:** https://jelve.netlify.app

---

## Features

- Clean multi-page layout  
- Fully responsive design  
- Simple admin login system  
- Editable homepage content  
- Contact information boxes  
- Organized and easy-to-modify structure  
- Fast deployment on Netlify

## Editing the Project

### 1. Edit Locally

Clone the repository and modify it using any editor (VS Code recommended):

git clone <YOUR_REPOSITORY_URL>
cd <YOUR_PROJECT_NAME>

This project is built with standard HTML, CSS, and JavaScript,
so no build tools are required — just open index.html in your browser.

### 2. Edit Directly on GitHub

Open the file you want to update

Click the Edit (pencil) button

Make your changes

Commit them to the repository

Your Netlify deploy will update automatically.
